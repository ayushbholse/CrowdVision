# CrowdVision

## Overview

CrowdVision is an AI-powered crowd monitoring and analytics mobile application developed as a Final Year BSc Information Technology project.

The application uses real-time face detection and crowd analysis techniques to monitor crowd density, generate reports, and provide actionable insights through an intuitive dashboard.

## Features

* Real-time Crowd Detection
* Face Detection using TensorFlow
* Crowd Density Monitoring
* Analytics Dashboard
* Emergency Alert System
* User Authentication
* Report Generation
* Data Storage using SQLite
* Cloud Storage Integration
* Notification System
* Crowd Heatmap Visualization
